How to use DAX Format Builder

1
Choose a display category, format style, decimal places and separator convention from the left sidebar.
2
Use the Options panel (right) to customise zero handling, negative display, thousands separator, trailing zeros, and currency symbol.
3
The format string updates live at the top. Press Copy and paste directly into Power BI.

Format string structure

positive ; negative ; zero

Each section is separated by a semicolon. This tool always generates all three parts so you have full control over how zeros appear.

Separator convention

The format string itself does not change — Power BI applies separators based on your report locale. The toggle adjusts the sample preview only so you can verify output looks correct for your region.
